## IMAGE GUIDELINES

### Quality Settings
- **JPG Quality:** 80% (good balance of quality/size)
- **Max file size:** 150KB for large images, 100KB for cards, 60KB for thumbnails
- **Format:** JPG for photos, PNG only if transparency needed

### Style Guidelines
- **Lighting:** Well-lit, professional industrial photography
- **Colors:** Should complement brand colors (blue #003087, teal #00B785)
- **Composition:** Subject centered or rule-of-thirds
- **No text overlays:** Keep images clean for flexibility
- **High resolution source:** Start with at least 2x the target size, then resize down

### Optimization Process
1. Resize to target dimensions
2. Export as JPG at 80% quality
3. Run through image optimizer (TinyPNG, ImageOptim, or Squoosh)
4. Verify file size is within limits
5. Test on mobile for loading speed
